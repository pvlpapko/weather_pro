# Политика конфиденциальности Weather Widget Pro

Приложение Weather Widget Pro не содержит рекламы, рекламных SDK и инструментов рекламного профилирования.

## Какие данные используются

Приложение может использовать:

- геолокацию устройства — чтобы определить текущий населённый пункт и получить прогноз погоды;
- выбранный пользователем город — чтобы показывать прогноз без геолокации;
- настройки приложения и виджетов — язык, тема, единицы измерения, прозрачность виджета, размер часов/погоды, выбранный диапазон прогноза;
- уведомления — для предупреждений о дожде и функции “Погода на экране блокировки / AOD”.

## Внешние сервисы

Для прогноза погоды приложение использует Open-Meteo.

Для поиска города приложение использует Open-Meteo Geocoding.

Для обратного определения названия населённого пункта приложение может использовать Android Geocoder, а также резервно OpenStreetMap Nominatim.

При запросах к погодным и геокодинговым сервисам может передаваться приблизительная координата или название города, необходимые для получения прогноза и определения населённого пункта.

## Что приложение не делает

Приложение:

- не продаёт пользовательские данные;
- не показывает рекламу;
- не собирает рекламный профиль;
- не передаёт данные рекламным сетям;

## Экран блокировки и AOD

Если пользователь включает настройку “Погода на экране блокировки / AOD”, приложение показывает постоянное тихое уведомление с погодой. Видимость этого уведомления на экране блокировки и Always On Display зависит от настроек Android и оболочки телефона.


## Синхронизация времени виджета

Часы на виджетах используют системное время Android. Для этого приложение не собирает и не передаёт дополнительные данные: отображение времени выполняется локально на устройстве.
