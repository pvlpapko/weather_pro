# Changelog

## 2.3.20 — versionCode 25

- Исправлена синхронизация времени на виджетах “Часы + погода” и “Часы по центру + погода”: часы переведены на системный `TextClock`, поэтому берут время напрямую из Android и обновляются вместе с временем телефона.
- Убрана ручная установка текста времени из `RemoteViews`, чтобы виджет не зависел от задержанных системных обновлений AppWidget.
- Увеличены погодные изображения на виджетах часов: солнце, луна, облака, дождь, снег, туман и гроза теперь заметнее.
- Увеличена кнопка-шестерёнка на виджетах, чтобы было проще попасть пальцем в настройки.
- Обновлены публикационные файлы: README, privacy-policy, fastlane metadata и changelog.
- `applicationId` сохранён: `com.ltm.weather`.
- Подпись сборки оставлена через существующий `signingConfigs.updateCompatible` и стабильный `app/dev-update-key.jks` / GitHub Secrets.

## 2.3.19 — versionCode 24

- Добавлены реалистичные погодные изображения на основных виджетах вместо системных эмодзи.
- Улучшена погодная анимация в приложении: солнце, луна, облака, дождь, снег, туман и гроза.
- Обновлены информационные файлы проекта под текущую функциональность.
